一、项目介绍
==============
  * 本项目为JavaWeb项目之客户管理系统，使用的技术是JSP+Servlet+Mysql<br>
<br>

功能介绍
-----------
* 添加客户：添加客户信息，其中包括姓名、性别、手机、邮箱和自我描述 <br>
* 查询客户：查询数据库中的所有客户信息，查询结果以列表分页形式展示，可以对展示列表的客户信息进行编辑和删除 <br>
* 高级搜索：可根据客户信息模糊搜索，搜索的结果以列表分页形式展示，可对展示列表的客户信息进行编辑和删除 <br>
